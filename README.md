# 🍽️ Restaurant Sales Analytics

> **An end-to-end data analytics project that transforms raw restaurant sales data into meaningful business insights using Python (Pandas) and Power BI.**

---

## 📌 Overview

This project demonstrates the complete data analytics workflow—from cleaning raw restaurant transaction data to creating an interactive Power BI dashboard. The dashboard helps analyze sales performance, profitability, customer behavior, and branch performance to support data-driven decision-making.

---

## 🎯 Objectives

- Clean and preprocess raw restaurant sales data
- Handle missing values, duplicates, and invalid records
- Analyze sales and profit trends
- Monitor branch and customer performance
- Build an interactive Power BI dashboard

---

## 🛠️ Tech Stack

- 🐍 Python
- 📊 Pandas
- 📈 Power BI
- ⚡ Power Query
- 📋 Microsoft Excel
- 📐 DAX

---

## 📂 Dataset

The dataset contains restaurant transaction details including:

- Order Information
- Customer Information
- Branch Details
- Food Categories
- Quantity & Sales
- Cost & Profit
- Payment Methods
- Customer Ratings
- Employee Performance

---

## 🧹 Data Cleaning

The raw dataset contained real-world data quality issues such as:

- Missing Values
- Duplicate Records
- Invalid Dates
- Incorrect Data Types
- Invalid Payment Methods
- Negative Quantities
- Outliers

These issues were identified and cleaned using **Python (Pandas)** before analysis.

## 📊 Dashboard Features

The Power BI dashboard includes:

- 💰 Total Sales

- 🏪 Branch-wise Performance
- 🍽️ Category-wise Sales
- ⭐ Customer Ratings
- 💳 Payment Method Analysis
